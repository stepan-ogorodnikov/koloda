## Internationalization (i18n)

- **Extract messages (demo)**: `nx run demo:lingui-extract --clean`
- **Compile messages (demo)**: `nx run demo:lingui-compile`
- **Extract messages (desktop)**: `nx run native-tauri-react:lingui-extract --clean`
- **Compile messages (desktop)**: `nx run native-tauri-react:lingui-compile`
- **Translation workflow**: Extract → Add translations to .po files → Compile
